ğŸ“Š Overview

PolyMetrics is a powerful Chrome extension for analyzing Envato products. It provides detailed revenue, profit, and sales analytics with interactive charts and customizable currency conversion.

ğŸ¯ Features

  • Revenue Analysis: Calculate total revenue, profit before/after tax with detailed breakdown
  • Multi-Currency Support: Convert amounts to 17+ currencies with real-time exchange rates
  • Interactive Charts: Bar, Line, Radar, and Polar Area charts for data visualization
  • Bookmark System: Save product pages with categories, search, and quick access panel
  • Quick Links: Save and access frequently used links (per product page)
  • Author Fee Calculation: Automatic calculation based on Envato's tiered fee structure (12.5% - 37.5%)
  • Collapsible Details: Toggle detailed fee breakdown tables with state memory
  • Description Editor: Live preview with markdown support for product descriptions

ğŸ’¡ How to Use

  1. Navigate to Product Page: Open any item page on CodeCanyon or ThemeForest
  2. View Analytics: The extension automatically displays metrics below the product header
  3. Adjust Tax Rate: Input your tax percentage in the toolbar (0-100%)
  4. Select Currency: Choose from dropdown to view amounts in different currencies
  5. Update Exchange Rate: Modify rates in Settings if needed
  6. Change Chart Type: Use dropdown in Revenue Breakdown chart (Bar, Line, Radar, Polar Area)
  7. Toggle Details: Click icon in top-right of fee block to show/hide detailed tables
  8. Manage Quick Links: Click ğŸ”— icon to add/remove frequently accessed links

âš™ï¸ Settings

Default Tax Rate: Set your standard tax percentage (e.g., 20% for corporate tax). This will be the default value when loading product pages.

Default Currency: Choose your preferred currency for display. All amounts will be converted and shown in this currency.

Currencies Management: Add, edit, or remove currencies with custom exchange rates. USD and VND are protected and cannot be removed.

Exchange Rates: All rates are relative to USD (1 USD = rate). For example, VND rate of 23000 means 1 USD = 23000 VND. Update regularly for accuracy.

ğŸ“ˆ Charts

  • Bar Chart: Compare revenue, profit, tax side by side (default view)
  • Line Chart: View trends with smooth curves, best for comparing values
  • Radar Chart: Multi-dimensional comparison in spider web format
  • Polar Area: Proportional distribution visualization
  • Pie Chart: Revenue distribution percentages (Net Profit, Tax, Fees)

ğŸ’¾ Data Persistence

All data is automatically saved to Chrome Storage and persists across browser sessions:

  • Global Settings: Tax rate, default currency, all currency definitions
  • Per-Page Data: Quick links (unique per product URL), toggle state
  • Storage Location: Chrome Local Storage (synced to your profile)

ğŸ†˜ Troubleshooting

  • Charts not showing: Refresh the page. Ensure Chart.js library is loaded (check browser console)
  • Toggle not working: Check console for errors. Ensure jQuery is loaded properly
  • Settings not saving: Verify Chrome Storage permission in manifest.json
  • Exchange rates outdated: Update rates manually in Settings tab
  • Quick Links not appearing: Ensure you're on the correct product page

ğŸ“ Important Notes

  • Author fee calculation follows Envato's tiered structure: 12.5% - 37.5% based on lifetime earnings
  • Buyer fee is fixed at $5 per item (Envato standard)
  • All financial calculations are in USD first, then converted to selected currency
  • Date parsing: "2 years ago" â†’ first day of that year; "10 months ago" â†’ first day of that month
  • Toggle state is remembered: if you close the fee breakdown table, it stays closed on reload
